# Magic 8 Ball App

A fun, interactive desktop application inspired by the famous fortune-telling Magic 8 Ball. Users can type any yes/no question about their future, pets, or life choices, and receive mystical, randomized predictions wrapped in a sleek modern interface.

## 🚀 Key Features

- **Randomized Mystical Logic:** Delivers diverse responses ranging from absolute certainties (*Yes*, *Absolutely*) to ambiguous predictions (*Maybe*, *The Universe is unsure*) and negatives.
- **Interactive UI Paradigm:** Designed with vibrant neon typography and custom layout configurations that align perfectly with the "magic" theme.
- **State Updates:** Instantly resets or updates text fields dynamically based on user interaction without freezing the window.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **GUI Framework:** CustomTkinter
- **Architecture:** Object-Oriented Programming (OOP) with modular configuration mappings.

## 💻 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Install dependencies:
   ```bash
   pip install customtkinter
   ```
3. Run the application:
   ```bash
   python main.py
   ```
